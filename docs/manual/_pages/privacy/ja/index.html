HDZap プライバシーポリシー

最終更新: 2026年5月

概要

HDZap はラップタイマーアプリです。無料版では全ての処理を端末上で行い、サーバーとは一切通信しません。オプションの HDZap Premium サブスクリプションを有効化した場合のみ、自然な音声で実況を読み上げるためにクラウド TTS サービスへラップ実況のテキストを送信します。データが iPhone を離れるのはこの経路のみです。

無料版 — 端末からデータは出ません

Premium を無効にしている状態（デフォルト）では、HDZap は以下のいずれも収集・送信・共有しません：

  • 個人情報（氏名、メールアドレス、電話番号など）
  • 位置情報
  • 利用統計やクラッシュレポート
  • 広告 ID やデバイス ID
  • あなたの iPhone 以外の場所にあるレース履歴

レース履歴（ラップタイム、レース記録）は iPhone の Application Support ディレクトリに保存されます。iCloud バックアップやローカルバックアップを有効にしていれば iOS の標準バックアップに含まれますが、開発者がそのデータを見ることはありません。

HDZap Premium — 送信される内容と送信先

HDZap Premium を有効化して Premium ボイスを選択すると、読み上げるフレーズごとに開発者運営のバックエンド（Cloudflare 上）にテキストが送信され、選択したボイスに応じて以下 3 つの音声合成（TTS）プロバイダーのいずれかに転送されます：

  • AWS Polly（Amazon Web Services, Inc.） — Polly Neural ボイス
  • Azure AI Speech（Microsoft Corporation） — Azure Neural ボイス
  • Cartesia（Cartesia, Inc.） — Cartesia Sonic ボイス

プロバイダーは音声を合成し、バックエンド経由で iPhone にストリーミングします。iPhone はその音声をスピーカー（または接続中の Bluetooth ヘッドホン）から再生します。

送信される内容

  • 実況する正確なフレーズ：ラップ番号、ラップタイム、「ベストラップ」のコール、カウントダウン番号など。例：ラップ3、12.34、ベストラップ や Lap 3, 12.34, best lap。
  • 選択したボイス ID、言語タグ、（該当する場合）Rate / Pitch の調整値。
  • サブスクライバーの場合：サブスクリプションが有効であることを示す Apple 署名済み JSON Web Signature（JWS） トークン。JWS にはサブスクリプションの商品 ID、有効期限、Original Transaction ID が含まれます。Apple ID・メールアドレス・氏名・デバイス ID は含まれません。

送信されない内容

  • レース履歴、現在の実況以外のラップデータ、その他の蓄積データ
  • 氏名、メールアドレス、Apple ID、その他個人を特定できる情報
  • 位置情報、IP ジオロケーション、デバイス ID

保持期間

バックエンドは合成済み音声を最大 30 日間キャッシュします（同じパラメータでのリクエストは再合成不要）。キャッシュには音声データのみが保存され、リクエストのメタデータは保持されません。キャッシュエントリは自動的に削除され、無期限に保管されることはありません。

各 TTS プロバイダーはリクエストを処理して音声を生成します。各社のデータ処理ポリシーが適用されます：

  • AWS Polly: aws.amazon.com/privacy
  • Azure AI Speech: privacy.microsoft.com/privacystatement
  • Cartesia: cartesia.ai/privacy

サブスクリプション課金

HDZap Premium は App Store 経由で販売されます。決済・請求・返金・本人確認はすべて Apple が処理します。開発者がクレジットカード情報・Apple ID・その他 Apple が管理する個人情報を見ることはありません。受け取るのは前述の匿名 Apple 署名済み JWS トークンのみです。解約方法：設定 → Apple ID → サブスクリプション、または paywall の「サブスク管理」ボタンから。

Bluetooth

HDZap は M5StickS3 ブリッジ機器との通信に Bluetooth Low Energy（BLE）を使用します。接続は端末間の直接通信で、開発者や第三者を経由しません。

サードパーティ SDK

HDZap には分析・広告・データ収集を行うサードパーティ SDK は含まれていません。Premium サブスクリプションの経路は iOS 標準の Apple StoreKit 2 と自社バックエンドのみを使用し、Firebase・Crashlytics などは利用していません。

13歳未満のお子様について

HDZap は 13 歳未満を対象としていません。当社は誰のデータも意図的に収集しないため、若年ユーザーが HDZap をインストールしても、その方のデータが端末から送信されることはありません。

本ポリシーの変更

データの流れが変更された場合（新しい TTS プロバイダーの追加、バックエンドの移転など）、本ページを更新します。冒頭の「最終更新」日が正式な変更日時です。
